# hi, i'm frederike

PhD researcher at the Geophysical Institute, University of Bergen (Norway).

I work on machine learning approaches for reconstructing atmospheric boundary layer turbulence profiles - bridging LES simulations and observational mast data using artificial neural networks.

**currently working on:**
- ANN-based reconstruction of turbulence statistics from LES (paper 1)
- adaptive scanning strategies for wind lidar measurements (paper 2)
- ROVER: drone-based atmospheric field campaigns
